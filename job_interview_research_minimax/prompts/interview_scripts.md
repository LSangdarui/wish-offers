## 二、高频问题回答框架（简化版）

### Q1: 对我们公司和产品了解多少？

**回答框架**：
> "Minimax 是国内领先的 AGI 公司，2026 年 1 月港股上市（0100.HK）。
> 
> 技术特色是全模态布局（文本/语音/视频/音乐），采用混合架构实现极致质价比。
> 
> 核心产品包括 M1/M2 系列大模型、海螺视频生成、MiniMax Agent。
> 
> 差异化优势是 73% 收入来自海外，多模态能力国内最强。
> 
> 战略方向是 2025-2026 年重点布局 ToB，发力金融/零售等行业。"

### Q2: 你觉得 Minimax 相比智谱/百度有什么优势？

**回答框架**：
> "Minimax 的差异化主要在四个方面：
> 
> 1. **全模态能力**：国内唯一同时做到文本、语音、视频、音乐四大赛道第一梯队的公司。
> 2. **极致性价比**：混合架构让推理成本大幅降低，API 价格比 OpenAI 低 50% 以上。
> 3. **全球化能力**：73% 收入来自海外，Talkie 全球 2.12 亿用户，出海经验丰富。
> 4. **独立第三方**：非大厂背景，客户无利益冲突顾虑。"

### Q3: 客户说"我们用智谱挺好的，为什么要换"怎么回应？

**LACE 法则**：
> **L - Listen（倾听）**：了解客户用智谱的场景和满意度。
> **A - Acknowledge（认可）**：认可智谱在文本基座模型上的实力。
> **C - Challenge/补充（提出新维度）**：
>   - 如果有多模态需求（视频/语音/音乐），只有 Minimax 能做。
>   - 如果考虑成本，同样调用量我们可以便宜 30%。
>   - 如果有出海业务，我们的多语言经验更有价值。
> **E - Explore（探索）**：邀请做 POC 对比测试，用数据说话。

### Q4: 你认为大模型在企业落地的最大挑战是什么？

**分层回答**：
> "我认为有五个层面的挑战：
> 
> **技术层面**：模型幻觉问题、与现有系统集成难度、效果不稳定。
> **成本层面**：算力成本高、ROI 难以量化。
> **安全层面**：数据隐私顾虑、合规要求（金融/政务需要本地化部署）。
> **组织层面**：员工抵触、AI 人才缺乏、流程需要重构。
> **认知层面**：期望过高、不知道哪些场景适合用 AI。
> 
> Minimax 的销售价值在于不只是卖 API，而是帮客户解决这些挑战。"

## 五、技术 Demo 准备脚本（可选）

### Python 调用 Minimax API 示例

```python
import requests

def call_minimax_api(prompt, model="MiniMax-M2"):
    """
    调用 Minimax API 生成文本
    面试前需注册开放平台获取 API Key
    """
    api_key = "your_api_key_here"
    url = "https://api.minimaxi.com/v1/text/chatcompletion"
    
    headers = {
        "Authorization": f"Bearer {api_key}",
        "Content-Type": "application/json"
    }
    
    data = {
        "model": model,
        "messages": [
            {"role": "system", "content": "你是一个专业的助手"},
            {"role": "user", "content": prompt}
        ],
        "temperature": 0.7,
        "max_tokens": 1000
    }
    
    try:
        response = requests.post(url, headers=headers, json=data)
        result = response.json()
        return result['choices'][0]['message']['content']
    except Exception as e:
        return f"调用出错: {str(e)}"

# 面试演示示例
if __name__ == "__main__":
    # 示例：生成商品描述
    result = call_minimax_api("为'无线降噪耳机'写一段吸引人的商品描述")
    print(result)
```
